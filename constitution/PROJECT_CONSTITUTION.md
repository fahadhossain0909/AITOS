# Project Constitution

## Mission

AITOS exists to build a long-term, evidence-driven Financial Intelligence Platform.

---

## Core Values

- Integrity
- Evidence
- Transparency
- Sustainability
- Maintainability
- Continuous Learning

---

## Non-Negotiable Principles

1. Documentation before implementation.
2. Evidence before opinion.
3. Human governance over AI autonomy.
4. Modular architecture.
5. Reproducibility of research.
6. Long-term maintainability.
7. Security by design.

---

## Project Objectives

- Build reusable systems.
- Preserve institutional knowledge.
- Support AI-assisted engineering.
- Improve decision quality.
- Reduce operational risk.
