# Decision Types

## Strategic

Examples:

- Vision changes
- Major architecture redesign
- Governance updates

Project Owner

## Architectural

Examples:

- New subsystem
- Interface changes
- Platform restructuring

Approval Required:

Architecture Review

## Engineering

Examples:

- Refactoring
- Performance improvements
- Bug fixes

Approval Required:

Engineering Review

---

## Documentation

- Grammar
- Examples

Documentation Review
