# AI Context Layer

Version: 1.0

---

## Purpose

The Context Layer provides structured project knowledge that enables AI coding assistants to understand AITOS without repeatedly scanning the entire repository.

This directory serves as the project's persistent working memory.

---

## Design Goals

- Reduce repeated explanations
- Improve AI reasoning quality
- Preserve engineering context
- Accelerate onboarding
- Support multiple AI providers

## Update Policy

Whenever architecture, standards, or project goals change, the corresponding context files should also be updated.

Context files are considered living documents.
