# AES-003 — Coding Standard

## Objective

Source code should prioritize readability, correctness, and maintainability.

---

## Rules

- Small functions
- Clear naming
- Consistent formatting
- No duplicated logic
- Self-documenting code where practical

---

## Mandatory

- Version control
- Code review
- Unit tests
- Documentation updates
- Static analysis where applicable

---

## Quality

Code should be:

- Simple
- Predictable
- Testable
- Maintainable
