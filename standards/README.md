# AITOS Engineering Standards (AES)

Version: 1.0

---

# Purpose

The AITOS Engineering Standards (AES) define the mandatory engineering practices followed throughout the project.

These standards ensure consistency, maintainability, traceability, and long-term sustainability.

Standards are mandatory.

Guidelines are recommendations.

---

| AES-002 | Architecture |
| AES-003 | Coding |

| AES-005 | Testing |
| AES-006 | Security |
| AES-007 | Deployment |

---

# Compliance

Every pull request should comply with the applicable standards.

Exceptions require documented justification.
