# Code of Conduct

## Our Commitment

AITOS is committed to creating a respectful, inclusive, and professional environment for everyone.

We welcome contributors regardless of background, experience, or identity.

---

# Expected Behavior

Contributors are expected to:

- Be respectful.
- Be constructive.
- Be evidence-driven.
- Accept feedback professionally.
- Help others learn.
- Respect different viewpoints.

---

# Unacceptable Behavior

The following behaviors are unacceptable:

- Harassment
- Personal attacks
- Discrimination
- Hate speech
- Trolling
- Intimidation
- Deliberate misinformation
- Spam

---

# Conflict Resolution

Disagreements should be resolved through:

- Evidence
- Technical discussion
- Documentation
- Architecture review

Personal arguments should never replace objective engineering decisions.

---

# Enforcement

Project maintainers may:

- Request changes
- Remove inappropriate content
- Reject pull requests
- Temporarily restrict participation
- Permanently ban repeated violations
